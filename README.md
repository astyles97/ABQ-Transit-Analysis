# ABQ Transit Analysis

A data analysis project seeking to answer the question if making ART free increased employment in Albuquerque, New Mexico from 2019 to 2026.

## The Question

Did making Albuquerque's public transit system, ART, free for all increase employment in the city? And can we see that increase more prominantly in certain sectors?

### Data Sources

Data is pulled from transit.dot.gov City of Albuquerque Transit Agency Profile and New Mexico Department of Workforce Solutions County Employment & Wages.

Links: \
https://www.transit.dot.gov/ntd/transit-agency-profiles/city-albuquerque \
